# PDF 割付くん (PDF Layout Tool)

ブラウザ上で動作する、PDF割り付け・加工ツールです。
2面割り付け、4面割り付け、右開き・左開きの変更などを、ファイルをアップロードすることなくクライアントサイド（ブラウザ内）だけで安全に処理します。

## 機能
- **2面 / 4面 割り付け**: A4用紙などに複数のページを配置してPDFを生成。
- **読み方向の指定**: 左→右（算数など）、右→左（国語など）に対応。
- **用紙・綴じ方向の指定**: 出力用紙の向き（縦/横）や両面印刷時の綴じ方向（短辺/長辺）を指定可能。
- **PWA対応**: Android / iOS にインストールして、オフラインでも一部機能を利用可能（初回アクセス後はキャッシュ利用）。

## 技術スタック
- **HTML5 / CSS3 / JavaScript (Vanilla)**
- **pdf-lib**: クライアントサイドでのPDF生成・加工
- **Lucide Icons**: アイコン表示
- **PWA (Progressive Web App)**: Service Worker, Manifest

## デプロイ方法
GitHubにプッシュ後、Vercelと連携するだけで自動的にデプロイされます。
`vercel.json` が含まれており、適切なキャッシュヘッダーなどが設定されます。
